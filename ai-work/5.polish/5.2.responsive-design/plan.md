# 5.2 Responsive Design — Plan

Perform a comprehensive responsive design pass across every screen targeting three breakpoints: mobile (375px), tablet (768px), and desktop (1024px+). Use Mantine's responsive props and CSS media queries to ensure touch targets are at least 44px, the canvas editor is usable with finger input, images are optimized for mobile bandwidth, and navigation adapts cleanly between bottom bar and sidebar layouts.
