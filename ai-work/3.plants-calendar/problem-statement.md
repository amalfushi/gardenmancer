# Milestone 3: Plants & Calendar

## Problem Statement

Gardenmancer needs a browsable plant database and a zone-aware planting calendar so users can discover what to plant, when to plant it, and how to care for it. This milestone creates the information layer that transforms raw plant data into actionable gardening guidance tailored to the user's USDA hardiness zone.

The plant database UI includes a searchable, filterable list of all plants (both seeded and scanned), detailed plant views showing spacing, sun, water, and growing requirements, and rich planting instructions. The calendar component calculates zone-specific planting windows — when to start seeds indoors, when to transplant, when to direct sow — based on the user's last frost date and each plant's requirements. The calendar UI presents this timeline visually so users can see at a glance what needs attention this week, this month, and this season.

The "add to garden" flow bridges this milestone with Milestone 4 — from any plant detail view, users can add the plant to one of their garden layouts, specifying quantity and preferred location. This creates the data link between the plant database and the garden editor, enabling the layout tool to know each plant's spacing and companion planting requirements.
