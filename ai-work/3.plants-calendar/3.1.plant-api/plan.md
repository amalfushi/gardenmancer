# 3.1 Plant API — Plan

Create Next.js API routes under `/api/plants` supporting full CRUD operations: `GET /api/plants` (list with search/filter query params for name, zone, sun, category), `GET /api/plants/[id]` (single plant detail), `POST /api/plants` (create), `PUT /api/plants/[id]` (update), and `DELETE /api/plants/[id]` (delete). All routes use the typed PlantStore wrapper.
