# 3.6 Add to Garden Flow — Plan

Add an "Add to Garden" action accessible from plant detail pages and the plant list (via a quick-action button). Present a modal where users select a target garden (or create a new one), specify plant quantity, and optionally note preferred placement. Persist the selection as a `GardenPlant` record linked to both the garden and plant, ready for placement in the Milestone 4 canvas editor.
