# 1.1 Next.js Init — Plan

Initialize a new Next.js 16 project with TypeScript, App Router, and pnpm as the package manager. Configure `tsconfig.json` with strict mode and path aliases (`@/` → `src/`), set up the initial `src/app/layout.tsx` and `src/app/page.tsx`, and verify the dev server starts cleanly with `pnpm dev`.
