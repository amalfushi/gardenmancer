# Milestone 8: UX Polish (User Feedback)

## Problem

User testing revealed several UX issues: missing north indicator on garden canvas, scan tab only supports camera (not upload), calendar can't filter by garden plants, nav icon sizing inconsistency, data doesn't persist between runs, and we need branding (mascot).

## Sub-milestones

1. **8.1 North Arrow** — Compass indicator on garden canvas
2. **8.2 Photo Upload** — Support file upload in addition to camera capture on scan tab
3. **8.3 Calendar Filter** — Filter calendar to only show plants in your gardens
4. **8.4 Nav Icon Fix** — Fix Plants/Browse icon being smaller than others
5. **8.5 Persistence Fix** — Ensure db.json persists between dev server restarts
6. **8.6 Mascot** — Generate a spooky green plant-mancer wizard for branding
